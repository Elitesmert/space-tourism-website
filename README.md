## Installation

Clone the project.

```console
git clone https://github.com/Elitesmert/space-tourism-website.git

```

```console
cd project-name


pnpm install

# Or with npm:

npm install

```

## Usage

Use the following command to start the project:

```console

pnpm dev

# Or with npm:

npm run dev

```
